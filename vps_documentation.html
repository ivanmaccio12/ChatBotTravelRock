  • 3002: ChatBotImprenta (PM2)
  • 3003: MuniOran (PM2)

  • 6379: Redis (Docker — solo localhost)
  • 8080: Evolution API (Docker)

4. Recomendaciones para Nuevos Bots

  • Ubicación: Clonar el código en /var/www/NombreDelProyecto.
  • Puertos disponibles: Desde 3006 en adelante.
  • APIs Node.js: pm2 start src/index.js --name "NombreBot" y luego pm2 save.
  • Frontends SPA (React/Vue): Crear un server.cjs con Express 4 y ejecutar con pm2 start server.cjs --name "NombreApp". No usar pm2 serve — tiene bugs con SPAs en Express 5.
  • Base de Datos: Reutilizar PostgreSQL en 127.0.0.1:5432. Crear una base de datos separada por proyecto (CREATE DATABASE nombre_db).
  • n8n: Crear webhooks con rutas únicas por proyecto (/webhook/nombre-proyecto/...).
